# 演習の進め方

---

## 演習概要

本演習では、1スプリントごとに1機能を追加しながら、複数回スプリントを回して開発を進めます。

以下のスプリントに対応するスプリントページが用意されているので、そちらを参照しながら演習を進めてください。

- チュートリアルスプリント
- 演習スプリント

チュートリアルスプリントは、講師と一緒に作業をすすめてもらい、スプリントの進め方や各工程で実施する内容を理解してもらいます。
演習スプリントは、チュートリアルスプリントで理解した内容をもとに、1人でスプリントを回してもらいます。

---

## スプリント内で実施する内容

要件定義と設計については、事前に用意されたドキュメントページを参照してください。
演習では、それ以外の活動を実施してもらいます。

| スプリント期間 | 開発工程                   | テスト工程               |
| -------------- | -------------------------- | ------------------------ |
| 前半           | 要件定義(ドキュメント参照) | 要件レビュー・テスト分析 |
| 前半〜中盤     | 設計(ドキュメント参照)     | 設計レビュー・テスト設計 |
| 中盤           | 実装                       | テスト実装               |
| 後半           | テスト                     | 探索的テスト             |
