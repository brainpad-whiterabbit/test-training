# 演習スプリント 2

## スプリント概要

このスプリントでは、エラー表示、表示桁数制限、計算結果桁数制限の機能を追加することを目指します。

## 要件レビュー

チュートリアルスプリント同様の手順でレビューを実施し、要件定義書にレビュー内容を記載してください。記載内容はcommit & pushしてください。

## テスト設計

チュートリアルスプリント同様の手順でテストケースを作成してください。テストケース表の変更内容はcommit & pushしてください。

| テスト観点              | テストケース                                         | 入力                              | 期待値                                     |
| ----------------------- | ---------------------------------------------------- | --------------------------------- | ------------------------------------------ |
| 計算結果桁数制限 (F-14) | 加算による表示範囲超過でエラーが表示されること       | 999999 + 1 を計算実行             | 入力メッセージ欄に "Overflow" と表示される |
| 計算結果桁数制限 (F-14) | 乗算による表示範囲超過でエラーが表示されること       | 999999 \* 2 を計算実行            | 入力メッセージ欄に "Overflow" と表示される |
| 計算結果桁数制限 (F-14) | 除算による表示範囲超過でエラーが表示されること       | 999999 / 0.1 を計算実行           | 入力メッセージ欄に "Overflow" と表示される |
| 表示桁数制限 (F-13)     | 整数部が最大6桁まで正しく入力できること              | 123456 と入力                     | 画面表示が "123456" になる                 |
| 表示桁数制限 (F-13)     | 整数部が6桁を超えた場合に追加の入力が無視されること  | 123456 の後にさらに 7 を入力      | 画面表示が "123456" のまま維持される       |
| 表示桁数制限 (F-13)     | 小数部が最大4桁まで正しく入力できること              | 0.1234 と入力                     | 画面表示が "0.1234" になる                 |
| 表示桁数制限 (F-13)     | 小数部が4桁を超えた場合に追加の入力が無視されること  | 0.1234 の後にさらに 5 を入力      | 画面表示が "0.1234" のまま維持される       |
| 表示桁数制限 (F-13)     | 整数6桁と小数4桁の最大10桁まで同時に入力できること   | 123456.1234 と入力                | 画面表示が "123456.1234" になる            |
| 表示桁数制限 (F-13)     | 最大10桁の状態でさらに入力した際に値が維持されること | 123456.1234 の後にさらに 5 を入力 | 画面表示が "123456.1234" のまま維持される  |
| エラー表示 (F-12)       | 0除算が実行された場合にエラーが表示されること        | 5 / 0 を計算実行                  | 入力メッセージ欄に "Error" と表示される    |
