# 要件定義

演習用に作成した電卓Webアプリの簡易的な要件定義書です。

## 1. システム概要

### 1.1 システム名

電卓Webアプリ

```text
+------------------------+
| 電卓Webアプリ           |
|    (式表示エリア)  12 + |
+------------------------+
|            (表示エリア) |
|                     0  |
+------------------------+
| MC | MR | M- | M+ | AC |
| CE | C  | ±  | %  | ÷  |
| 7  | 8  | 9  | x  | -  |
| 4  | 5  | 6  | +  | =  |
| 1  | 2  | 3  | 0  | .  |
+------------------------+
```

### 1.2 目的

利用者がブラウザ上で四則演算を簡単に行えるWebアプリを提供する。

### 1.3 対象利用者

- ブラウザ上で簡単な計算を行いたい一般利用者

## 3. 機能要件

レビュー反映と対応スプリントは演習上の都合で存在

|    No | 分類   | 機能名         | 要件内容                                                     | レビュー点 | 対応スプリント           |
| ----: | ------ | -------------- | ------------------------------------------------------------ | ---------- | ------------------------ |
| F-001 | 入力   | 数字入力       | 0〜9の数字をボタン操作で入力できる                           |            | 実装済み                 |
| F-002 | 入力   | 小数点入力     | 「.」ボタン押下時に小数点を入力できる                        |            | 実装済み                 |
| F-003 | 入力   | 符号反転       | 「±」ボタン押下時に現在入力中の数値の正負を切り替える        |            | 実装済み                 |
| F-004 | 入力   | 入力制御       | 演算子の連続入力など不正な入力を防止する                     |            | 実装済み                 |
| F-005 | 演算   | 計算実行       | 「=」ボタン押下時に計算結果を表示する                        |            | 実装済み                 |
| F-006 | 演算   | 四則演算       | 加算、減算、乗算、除算ができる                               |            | チュートリアルスプリント |
| F-007 | 演算   | パーセント計算 | 「%」ボタン押下時に入力値を百分率として計算できる            |            | 演習スプリント1          |
| F-008 | エラー | エラー表示     | 0除算など計算不能な場合はエラーメッセージを表示する          |            | 演習スプリント1          |
| F-009 | クリア | Clear機能      | 「C」ボタン押下時に現在入力中の数値のみをクリアする          |            | 演習スプリント2          |
| F-010 | クリア | ClearEntry機能 | 「CE」ボタン押下時に現在入力中の演算単位をクリアする         |            | 演習スプリント2          |
| F-011 | クリア | AllClear機能   | 「AC」ボタン押下時に入力内容、計算結果、計算状態を初期化する |            | 演習スプリント2          |
| F-012 | メモリ | メモリ保存加算 | 「M+」ボタン押下時に現在表示値をメモリへ加算する             |            | 演習スプリント3          |
| F-013 | メモリ | メモリ保存減算 | 「M-」ボタン押下時に現在表示値をメモリから減算する           |            | 演習スプリント3          |
| F-014 | メモリ | メモリ呼出     | 「MR」ボタン押下時にメモリ保存値を表示する                   |            | 演習スプリント3          |
| F-015 | メモリ | メモリ消去     | 「MC」ボタン押下時にメモリ保存値を削除する                   |            | 演習スプリント3          |

## 4. 非機能要件

|     No | 分類         | 要件内容                                 | レビュー点 | 対応スプリント  |
| -----: | ------------ | ---------------------------------------- | ---------- | --------------- |
| NF-001 | 性能         | ボタン押下後、画面表示を速やかに更新する |            | 実装済み        |
| NF-002 | 対応ブラウザ | Chromeブラウザで利用できる               |            | 実装済み        |
| NF-004 | 操作性       | 主要ボタンを視認しやすく配置する         |            | 演習スプリント4 |
| NF-005 | 可用性       | ページ再読み込み後もアプリを再利用できる |            | 実装済み        |
| NF-006 | セキュリティ | 入力値としてスクリプトを実行しない       |            | 実装済み        |
| NF-007 | 保守性       | 機能ごとに処理を分割し修正しやすくする   |            | 実装済み        |

## 5. 受入条件

| 機能               | 条件               | 操作                          | 期待結果                        | レビュー点 | 対応スプリント           |
| ------------------ | ------------------ | ----------------------------- | ------------------------------- | ---------- | ------------------------ |
| 初期表示           | 初回アクセス時     | 画面を表示する                | `0` が表示される                |            | 実装済み                 |
| 数字入力           | 初期状態           | `1` を押下する                | `1` が表示される                |            | 実装済み                 |
| 数字入力           | 数字表示中         | `1` → `2` を押下する          | `12` が表示される               |            | 実装済み                 |
| 小数点入力         | 整数入力済         | `1 . 5` を押下する            | `1.5` が表示される              |            | チュートリアルスプリント |
| 小数点制御         | 小数点入力済       | `1 . . 5` を押下する          | `1.5` が表示される              |            | 実装済み                 |
| 演算子優先順位     | 複数演算入力時     | `2 + 3 × 4 =` を押下する      | `14` が表示される               |            | 実装済み                 |
| 連続計算           | 計算結果表示後     | `1 + 2 = + 3 =` を押下する    | `6` が表示される                |            | 実装済み                 |
| 加算               | 初期状態           | `1 + 2 =` を押下する          | `3` が表示される                |            | 実装済み                 |
| 減算               | 初期状態           | `5 - 2 =` を押下する          | `3` が表示される                |            | チュートリアルスプリント |
| 乗算               | 初期状態           | `3 × 4 =` を押下する          | `12` が表示される               |            | チュートリアルスプリント |
| 除算               | 初期状態           | `8 ÷ 2 =` を押下する          | `4` が表示される                |            | チュートリアルスプリント |
| 小数計算           | 初期状態           | `1.5 + 2.5 =` を押下する      | `4` が表示される                |            | 実装済み                 |
| 負数計算           | 初期状態           | `5 - 10 =` を押下する         | `-5` が表示される               |            | 実装済み                 |
| イコール連打       | 計算実行後         | `2 + 3 = =` を押下する        | `8` が表示される                |            | 実装済み                 |
| 先頭ゼロ制御       | 初期状態           | `0 0 1` を押下する            | `1` が表示される                |            | 実装済み                 |
| クリア             | 値入力済           | `C` を押下する                | 現在入力値のみクリアされる      |            |                          |
| オールクリア       | 計算途中           | `AC` を押下する               | 入力値・計算状態が初期化される  |            |                          |
| ゼロ除算           | 除算時             | `1 ÷ 0 =` を押下する          | `ERROR` が表示される            |            | チュートリアルスプリント |
| 桁数制限           | 最大桁数到達時     | さらに数字を入力する          | 追加入力されない                |            |                          |
| 演算子連続入力     | 演算子入力済       | `1 + - 2 =` を押下する        | 最新演算子で上書きされる        |            |                          |
| 空入力演算         | 初期状態           | `+` を押下する                | エラーにならない                |            |                          |
| 結果後数字入力     | 計算結果表示後     | `1 + 2 =` 後に `5` を押下する | 新規入力として `5` が表示される |            |                          |
| 結果後演算入力     | 計算結果表示後     | `1 + 2 = + 5 =` を押下する    | `8` が表示される                |            |                          |
| レスポンシブ対応   | スマホ表示時       | 画面幅320pxで表示する         | 横スクロールが発生しない        |            |                          |
| ボタン操作性       | 通常表示時         | 任意ボタンを押下する          | 押下状態が視覚的に分かる        |            |                          |
| 表示崩れ防止       | 長い数値表示時     | 最大桁数を入力する            | 表示領域からはみ出さない        |            |                          |
| ブラウザ対応       | 対応ブラウザ利用時 | Chromeで利用する              | 正常動作する                    |            |                          |
| アクセシビリティ   | キーボード操作時   | Tabキーで移動する             | 全ボタンへフォーカス移動できる  |            |                          |
| エラーハンドリング | 異常操作時         | 不正入力を行う                | 画面がフリーズしない            |            |                          |
| パフォーマンス     | 通常利用時         | 画面へアクセスする            | 3秒以内に初期表示される         |            |                          |
| リロード対応       | 計算途中           | ブラウザ更新を行う            | 初期状態で表示される            |            |                          |
