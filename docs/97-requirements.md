# 要件定義

演習用に作成した電卓Webアプリの簡易的な要件定義書

## 1. システム概要

2つの数値を入力し、四則演算やパーセント計算ができる電卓Webアプリを提供する。

### 1.1 システム名

電卓Webアプリ

### 1.2 目的

ユーザがブラウザ上で基本的な電卓機能を行えるWebアプリを提供する。

### 1.3 対象利用者

ブラウザ上で簡単な計算を行いたいユーザ

---

## 2. 画面

```text
+------------------------+
| 電卓Webアプリ           |
|        (式表示欄)  12 + |
+------------------------+
|           (入力欄)  0  |
+------------------------+
| CE | C  | ±  | %  | ÷  |
| 7  | 8  | 9  | x  | -  |
| 4  | 5  | 6  | +  | =  |
| 1  | 2  | 3  | 0  | .  |
+------------------------+
```

- 式表示欄: 入力された数値や演算子を表示する
- 入力欄: 現在入力中の数値や計算結果を表示する
- ボタン: 数字、演算子、クリア系の入力を受け付ける

## 3. 状態遷移

電卓上の状態遷移を定義する。

```mermaid
stateDiagram-v2
    [*] --> 入力待ち

    入力待ち --> 入力1 : 数値入力
    入力1 --> 入力1 : 数値入力, 符号反転, 入力クリア
    入力1 --> 入力待ち : クリア
    入力1 --> 入力2 : 演算子入力

    入力2 --> 入力2 : 数値入力, 符号反転, 入力クリア
    入力2 --> 入力待ち : クリア, 計算実行

    入力2 --> エラー : 計算実行(実行不能)
    エラー --> 入力待ち : クリア, 入力クリア
```

- 状態
    - 入力待ち: 数値入力を待っている状態
    - 入力1: 入力1を受け付けている状態
    - 入力2: 入力2を受け付けている状態
    - エラー: 計算不能な状態
- 行動
    - 数値入力: 数字ボタンや小数点の入力
    - 符号反転: ±ボタンの入力
    - 演算子入力: +, -, x, ÷ ボタンの入力
    - クリア: Cボタンの入力
    - 入力クリア: CEボタンの入力
    - 計算実行: =, % ボタンの入力

## 4. 機能要件

機能要件として、システムの機能・振る舞いを定義する。

|  No. | 分類     | 機能名               | 要件内容                                                                                                                                                                                                  | レビュー点 | 対応スプリント           |
| ---: | -------- | -------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ---------- | ------------------------ |
| F-01 | 入力     | 数字入力             | 数字ボタン（0〜9）押下時に、その値を入力欄に表示する。連続で数字を入力すると桁数が増えて入力される。                                                                                                      |            | 対応済み                 |
| F-02 | 入力     | 小数点入力           | 「.」ボタン押下時に、入力欄へ小数点を追加する。ただし、連続で少数点を入力することはできない                                                                                                               |            | 対応済み                 |
| F-03 | 入力     | 符号反転             | 「±」ボタン押下時に、入力欄の数値の符号を反転する。                                                                                                                                                       |            | 対応済み                 |
| F-04 | 入力     | 式表示               | 入力2の状態で、入力1と演算子を式表示欄に表示する。 計算実行が成功すると入力1と演算子と入力2を式表示欄に表示する。それ以外の状態では何も表示しない。                                                       |            | 対応済み                 |
| F-05 | 入力     | 入力制御             | 演算子をを連続で入力した際、最後に入力された演算子のみを有効とする。                                                                                                                                      |            | 対応済み                 |
| F-06 | 計算     | 計算実行             | 「=」ボタン押下時に、入力された式を計算し、入力欄に結果を表示する。                                                                                                                                       |            | 対応済み                 |
| F-07 | 演算     | 加算                 | 「+」ボタン押下時に、加算の演算子を式に登録し、計算実行すると入力された2つの数値を加算する。                                                                                                              |            | 対応済み                 |
| F-08 | 演算     | 減算                 | 「-」ボタン押下時に、減算の演算子を式に登録し、計算実行すると入力された2つの数値を減算する。                                                                                                                                       |            | チュートリアルスプリント |
| F-09 | 演算     | 乗算                 | 「x」button押下時に、乗算の演算子を式に登録し、計算実行すると入力された数値を乗算する。                                                                                                                   |            | 演習スプリント1          |
| F-10 | 演算     | 除算                 | ユーザが「÷」ボタン押下時に、除算の演算子を式に登録し、計算実行すると入力された2つの数値を乗算する。                                                                                                      |            | 演習スプリント1          |
| F-11 | 演算     | 小数演算             | 入力された2つの数値が小数を含む場合でも演算する。                                                                                                                                                         |            | 対応済み                 |
| F-12 | エラー   | エラー表示           | 0除算など計算不能な計算が実行された場合に、入力メッセージ欄にエラーメッセージ(Error)を表示する。                                                                                                          |            | 演習スプリント2          |
| F-13 | 制限     | 表示桁数制限         | 最大10桁(整数部最大6桁、小数部最大4桁)まで入力を受け付ける。上限を超えた場合 、表示の値を維持する。aaaaaaaa                                                                                               |            | 演習スプリント2          |
| F-14 | 制限     | 計算結果桁数制限     | 計算結果が表示桁数を超えた場合に、エラーメッセージ(Overflow)を入力欄に表示する。                                                                                                                          |            | 演習スプリント2          |
| F-15 | クリア   | クリア               | 「C」ボタン押下時に、入力1・入力2を含む入力中の計算式、計算結果、演算状態を初期化し、入力欄に `0` を表示する。                                                                                            |            | 対応済み                 |
| F-16 | クリア   | 入力クリア           | 「CE」ボタン押下時に、現在入力中の値を `0` にリセットする。入力2の状態で押下した場合、入力1と演算子は保持する。                                                                                           |            | 対応済み                 |
| F-17 | 特殊計算 | パーセント計算実行   | 入力2の状態の時の「%」ボタン押下時に入力2の数値を百分率として式を計算して、入力欄に結果を表示する。また、入力2以外の状態で、「%」ボタン押下時にクリアされる。                                             |            | 演習スプリント3          |
| F-18 | 特殊計算 | パーセント計算ルール | 入力が`[入力1, + または -, 入力2, %]`の時、 `入力1 演算子 (左辺 × 右辺 ÷ 100))` の計算が実行される。また、入力が`[入力1, x または /, 入力2, %]`の時、 `入力1 計算記号 (入力2 ÷ 100)` の計算が実行される。 |            | 演習スプリント3          |
