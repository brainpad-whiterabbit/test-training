# 要件定義

演習用に作成した電卓Webアプリの簡易的な要件定義書です。

## 1. システム概要

### 1.1 システム名

電卓Webアプリ

```text
+------------------------+
| 電卓Webアプリ           |
|    (式表示エリア)  12 + |
+------------------------+
|            (表示エリア) |
|                     0  |
+------------------------+
| MC | MR | M- | M+ | AC |
| CE | C  | ±  | %  | ÷  |
| 7  | 8  | 9  | x  | -  |
| 4  | 5  | 6  | +  | =  |
| 1  | 2  | 3  | 0  | .  |
+------------------------+
```

### 1.2 目的

ユーザがブラウザ上で基本的な電卓機能を行えるWebアプリを提供する。

### 1.3 対象利用者

ブラウザ上で簡単な計算を行いたいユーザ

---

## 3. 機能要件

機能要件として、システムの機能・振る舞いを定義する。

|   No | 分類   | 機能名           | 要件内容                                                                                               | レビュー点 | 対応スプリント           |
| ---: | ------ | ---------------- | ------------------------------------------------------------------------------------------------------ | ---------- | ------------------------ |
| F-01 | 入力   | 数字入力         | 数字ボタン（0〜9）押下時に、押下された数値を入力欄へ追加し、入力内容を表示する。                       |            | 対応済み                 |
| F-02 | 入力   | 小数点入力       | 「.」ボタン押下時に、現在入力中の数値へ小数点を追加し、入力内容を表示する。                            |            | 対応済み                 |
| F-03 | 入力   | 符号反転         | 「±」ボタン押下時に、現在入力中の数値の符号を反転し、反転後の値を表示する。                            |            | 対応済み                 |
| F-04 | 入力   | 入力制御         | 演算子入力の後に別の演算子が入力された場合に、後続の演算子入力を受け付けず、直前の入力内容を保持する。 |            | 対応済み                 |
| F-05 | 入力   | 計算実行         | 「=」ボタン押下時に、入力された計算式を演算し、計算結果を表示する。                                    |            | 対応済み                 |
| F-07 | 演算   | パーセント計算   | 「%」ボタン押下時に、現在入力中の数値を百分率として演算し、計算結果を表示する。                        |            | 対応済み                 |
| F-06 | 演算   | 加算             | 入力された2つの数値を加算(+)し、計算結果を表示する。                                                   |            | 対応済み                 |
| F-06 | 演算   | 減算             | 入力された2つの数値を減算(-)し、計算結果を表示する。                                                   |            | チュートリアルスプリント |
| F-06 | 演算   | 乗算             | 入力された2つの数値を乗算(x)し、計算結果を表示する。                                                   |            | 演習スプリント1          |
| F-06 | 演算   | 除算             | 入力された2つの数値を除算(/)し、計算結果を表示する。                                                   |            | 演習スプリント1          |
| F-08 | エラー | エラー表示       | 0除算など計算不能な演算が実行された場合に、エラーメッセージ(Error)を表示する。                         |            | 演習スプリント1          |
| F-08 | 制限   | 表示桁数制限     | 最大12桁(整数部最大10桁、小数部最大8桁)まで入力を受け付ける。上限を超えた場合 、表示の値を維持する。   |            | 演習スプリント2          |
| F-08 | 制限   | 計算結果桁数制限 | 演算結果が表示桁数を超えた場合に、エラーメッセージ(Overflow)を表示する。                               |            | 演習スプリント2          |
| F-09 | クリア | 入力欄消去       | 「C」ボタン押下時に、現在入力中の数値を削除し、入力欄を初期状態にする。                                |            | 対応済み                 |
| F-10 | クリア | 入力値消去       | 「CE」ボタン押下時に、現在入力中の演算対象値を削除し、それ以前の入力内容を保持する。                   |            | 演習スプリント3          |
| F-11 | クリア | 全消去           | 「AC」ボタン押下時に、入力内容・計算結果・演算状態・メモリ状態を初期化する。                           |            | 演習スプリント3          |
| F-12 | メモリ | メモリ保存加算   | 「M+」ボタン押下時に、現在表示中の値をメモリ値へ加算し、加算後の値を保持する。                         |            | 演習スプリント4          |
| F-13 | メモリ | メモリ保存減算   | 「M-」ボタン押下時に、現在表示中の値をメモリ値から減算し、減算後の値を保持する。                       |            | 演習スプリント4          |
| F-14 | メモリ | メモリ呼出       | 「MR」ボタン押下時に、メモリに保持された値を入力欄へ表示する。                                         |            | 演習スプリント4          |
| F-15 | メモリ | メモリ消去       | 「MC」ボタン押下時に、メモリに保持された値を削除する。                                                 |            | 演習スプリント4          |

## 4. 非機能要件

非機能要件として、機能以外でどのような品質・制約で動作するか定義する。
(演習用に抜粋)

|    No | 分類         | 要件内容                                                                   | レビュー点 | 対応スプリント  |
| ----: | ------------ | -------------------------------------------------------------------------- | ---------- | --------------- |
| NF-01 | 性能         | ボタン押下後1秒以内に画面表示を更新する                                    |            | 対応済み        |
| NF-02 | 対応ブラウザ | Google Chromeで利用できる                                                  |            | 対応済み        |
| NF-03 | 操作性       | 数字(青)・演算子(オレンジ)・クリア系ボタン(赤)が色分けされており識別できる |            | 演習スプリント5 |
| NF-04 | 可用性       | ページ再読み込み後に初期状態で操作を開始できる                             |            | 対応済み        |

## 5. 受入条件

|     No | 機能           | 条件                   | 操作                                          | 期待結果                                                               | レビュー点 | 対応スプリント           |
| -----: | -------------- | ---------------------- | --------------------------------------------- | ---------------------------------------------------------------------- | ---------- | ------------------------ |
| AC-001 | 数字入力       | 初期状態               | `1` を押下する                                | 入力欄に `1` が表示される                                              |            | 対応済み                 |
| AC-002 | 数字入力       | 数字入力済             | `1` → `2` を押下する                          | 入力欄に `12` が表示される                                             |            | 対応済み                 |
| AC-003 | 小数点入力     | 整数入力済             | `1 . 5` を押下する                            | 入力欄に `1.5` が表示される                                            |            | 対応済み                 |
| AC-004 | 符号反転       | 数値入力済             | `5` → `±` を押下する                          | 入力欄に `-5` が表示される                                             |            | 対応済み                 |
| AC-005 | 入力制御       | 演算子入力済           | `1 + -` を押下する                            | 後続の `-` は受け付けられず、直前の入力内容 `1 +` が保持される         |            | 対応済み                 |
| AC-006 | 計算実行       | 計算式入力済           | `1 + 2 =` を押下する                          | 計算結果として `3` が表示される                                        |            | 対応済み                 |
| AC-007 | 加算           | 初期状態               | `1 + 2 =` を押下する                          | 計算結果として `3` が表示される                                        |            | 対応済み                 |
| AC-008 | 減算           | 初期状態               | `5 - 2 =` を押下する                          | 計算結果として `3` が表示される                                        |            | チュートリアルスプリント |
| AC-009 | 乗算           | 初期状態               | `3 × 4 =` を押下する                          | 計算結果として `12` が表示される                                       |            | 演習スプリント1          |
| AC-010 | 除算           | 初期状態               | `8 ÷ 2 =` を押下する                          | 計算結果として `4` が表示される                                        |            | 演習スプリント1          |
| AC-011 | パーセント計算 | 数値入力済             | `50 %` を押下する                             | 現在入力中の数値が百分率として演算され、`0.5` が表示される             |            | 対応済み                 |
| AC-012 | エラー表示     | 0除算の計算式入力済    | `1 ÷ 0 =` を押下する                          | 計算不能であることを示すエラーメッセージが表示される                   |            | 演習スプリント1          |
| AC-013 | 入力欄消去     | 数値入力済             | `123` → `C` を押下する                        | 現在入力中の数値が削除され、入力欄が初期状態になる                     |            | 対応済み                 |
| AC-014 | 入力値消去     | 演算対象値入力済       | `12 + 34` → `CE` を押下する                   | 現在入力中の演算対象値 `34` が削除され、`12 +` は保持される            |            | 演習スプリント3          |
| AC-015 | 全消去         | 計算途中またはメモリ有 | `12 + 34` とメモリ値を保持後、`AC` を押下する | 入力内容・計算結果・演算状態・メモリ状態が初期化される                 |            | 演習スプリント3          |
| AC-016 | メモリ保存加算 | 数値表示中             | `10` → `M+` を押下する                        | 表示中の `10` がメモリ値へ加算され、加算後の値が保持される             |            | 演習スプリント4          |
| AC-017 | メモリ保存減算 | メモリ値と数値表示あり | メモリ値 `10` の状態で `3` → `M-` を押下する  | 表示中の `3` がメモリ値から減算され、メモリ値として `7` が保持される   |            | 演習スプリント4          |
| AC-018 | メモリ呼出     | メモリ値保持済         | メモリ値 `7` の状態で `MR` を押下する         | 入力欄にメモリ値 `7` が表示される                                      |            | 演習スプリント4          |
| AC-019 | メモリ消去     | メモリ値保持済         | `MC` を押下する                               | メモリに保持された値が削除される                                       |            | 演習スプリント4          |
| AC-020 | 表示更新性能   | 通常操作時             | 任意のボタンを押下する                        | ボタン押下後1秒以内に画面表示が更新される                              |            | 対応済み                 |
| AC-021 | 対応ブラウザ   | Google Chrome利用時    | 電卓Webアプリを操作する                       | Google Chromeで各機能を利用できる                                      |            | 対応済み                 |
| AC-022 | 操作性         | 通常表示時             | 画面上のボタンを確認する                      | 数字ボタンは青、演算子ボタンはオレンジ、クリア系ボタンは赤で識別できる |            | 演習スプリント5          |
| AC-023 | リロード対応   | 計算途中               | ブラウザのページ再読み込みを行う              | 初期状態で操作を開始できる                                             |            | 対応済み                 |
