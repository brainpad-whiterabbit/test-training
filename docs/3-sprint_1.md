# 演習スプリント 1

## スプリント概要

このスプリントでは、乗算と除算の機能を追加することを目指します。

## 要件レビュー

チュートリアルスプリント同様の手順でレビューを実施し、要件定義書にレビュー内容を記載してください。記載内容はcommit & pushしてください。

## テスト設計

チュートリアルスプリント同様の手順でテストケースを作成してください。テストケース表の変更内容はcommit & pushしてください。

| テスト観点 | テストケース             | 入力         | 期待値              |
| ---------- | ------------------------ | ------------ | ------------------- |
| 乗算       | 正数同士の乗算ができる   | 2 \* 2       | 4                   |
| 乗算       | 負数同士の乗算ができる   | (-2) \* (-2) | 4                   |
| 乗算       | 異符号同士の乗算ができる | 2 \* (-2)    | -4                   |
| 乗算       | 0を含む乗算ができる      | 2 \* 0       | 0                   |
| 乗算       | 小数を含む乗算ができる   | 2 \* 0.25    | 0.5                 |
| 除算       | 正数同士の除算ができる   | 2 / 2        | 1                   |
| 除算       | 負数同士の除算ができる   | (-2) / (-2)  | 1                   |
| 除算       | 異符号同士の除算ができる | (-2) / 1     | -2                  |
| 除算       | 0を含む除算ができない    | 1 / 0        | DivisionByZeroError |
| 除算       | 小数を含む除算ができる   | 0.5 / 2      | 0.25                |
