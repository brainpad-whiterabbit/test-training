# 演習スプリント 1

## スプリント概要

このスプリントでは、乗算と除算の機能を追加することを目指します。

## 要件レビュー

チュートリアルスプリント同様の手順でレビューを実施し、要件定義書にレビュー内容を記載してください。記載内容はcommit & pushしてください。

## テスト設計

チュートリアルスプリント同様の手順でテストケースを作成してください。テストケース表の変更内容はcommit & pushしてください。

| テスト観点 | テストケース                         | 入力                 | 期待値   |
| ---------- | ------------------------------------ | -------------------- | -------- |
| 乗算       | 負数同士の乗算ができる               | -2 \* (-3)           | 6        |
| 乗算       | 負数と正数の乗算ができる             | -2 \* 3              | -6       |
| 乗算       | 正数と負数の乗算ができる             | 2 \* (-3)            | -6       |
| 乗算       | 正数同士の乗算ができる               | 2 \* 3               | 6        |
| 乗算       | 0を含む乗算ができる                  | 999999 \* 0          | 0        |
| 乗算       | 最大値を超える乗算ができる           | 999999 \* 2          | Overflow |
| 乗算       | 最小値を下回る乗算ができる           | -999999 \* 2         | Overflow |
| 乗算       | 最大値同士の乗算ができる             | 999999 \* 999999     | Overflow |
| 乗算       | 最小値同士の乗算ができる             | -999999 \* (-999999) | Overflow |
| 除算       | 負数同士の除算ができる               | -6 / (-2)            | 3        |
| 除算       | 負数から正数を除算できる             | -6 / 2               | -3       |
| 除算       | 正数から負数を除算できる             | 6 / (-2)             | -3       |
| 除算       | 正数同士の除算ができる               | 6 / 2                | 3        |
| 除算       | 割り切れない除算ができる             | 3 / 2                | 1.5      |
| 除算       | 0を割る除算ができる                  | 0 / 5                | 0        |
| 除算       | 0で割る除算（ゼロ除算）ができる      | 5 / 0                | Error    |
| 除算       | 最大値を含む除算ができる             | 999999 / 1           | 999999   |
| 除算       | 最小値を含む除算ができる             | -999999 / 1          | -999999  |
| 除算       | 最小値を負数で除算して最大値を超える | -999999 / (-0.5)     | Overflow |
