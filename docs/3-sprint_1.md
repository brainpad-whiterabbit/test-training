# 演習スプリント 1

## スプリント概要

このスプリントでは、乗算と除算の機能を追加することを目指します。

## 要件レビュー

チュートリアルスプリント同様の手順でレビューを実施し、要件定義書にレビュー内容を記載してください。記載内容はcommit & pushしてください。

## テスト設計

チュートリアルスプリント同様の手順でテストケースを作成してください。テストケース表の変更内容はcommit & pushしてください。

| テスト観点 | テストケース                       | 入力             | 期待値 |
| ---------- | ---------------------------------- | ---------------- | ------ |
| 乗算       | 負数同士の乗算ができる             | -2 \* (-3)       | 6      |
| 乗算       | 負数と正数の乗算ができる           | -2 \* 3          | -6     |
| 乗算       | 正数同士の乗算ができる             | 2 \* 3           | 6      |
| 乗算       | 0を含む乗算ができる                | 5 \* 0           | 0      |
| 乗算       | 1を含む乗算ができる                | 5 \* 1           | 5      |
| 乗算       | 小数を含む乗算ができる             | 2.5 \* 2.0       | 5.0    |
| 乗算       | 負の小数を含む乗算ができる         | -1.5 \* 2.0      | -3.0   |
| 除算       | 負数同士の除算ができる             | -6 / (-2)        | 3      |
| 除算       | 負数を正数で除算できる             | -6 / 2           | -3     |
| 除算       | 正数を負数で除算できる             | 6 / (-2)         | -3     |
| 除算       | 正数同士の除算ができる             | 6 / 2            | 3      |
| 除算       | 0を含む除算ができる                | 0 / 5            | 0      |
| 除算       | 小数を含む除算ができる             | 5.0 / 2.0        | 2.5    |
| 除算       | 負の小数を含む除算ができる         | -3.0 / 2.0       | -1.5   |
| 除算       | 0で除算するとエラーになる          | 5 / 0            | Error  |
| 小数演算   | 小数の乗算と除算を組み合わせできる | 2.5 \* 2.0 / 2.0 | 2.5    |
| 小数演算   | 複数の小数演算を続けてできる       | 1.5 + 2.5        | 4.0    |
| 小数演算   | 浮動小数点の誤差が適切に処理される | 0.1 \* 3         | 0.3    |
