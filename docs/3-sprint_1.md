# 演習スプリント 1

## スプリント概要

このスプリントでは、乗算と除算の機能を追加することを目指します。

## 要件レビュー

チュートリアルスプリント同様の手順でレビューを実施し、要件定義書にレビュー内容を記載してください。記載内容はcommit & pushしてください。

## テスト設計

チュートリアルスプリント同様の手順でテストケースを作成してください。テストケース表の変更内容はcommit & pushしてください。

| テスト観点   | テストケース                 | 入力        | 期待値 |
| ------------ | ---------------------------- | ----------- | ------ |
| 乗算　　　　 | 正の数同士の乗算ができる     | 2 x 3       | 6      |
| 乗算         | 正の数と負の数の乗算ができる | 2 x (-3)    | -6     |
| 乗算         | 負の数同士の乗算ができる     | (-2) x (-3) | 6      |
| 乗算         | 0を含めた乗算ができる        | (-2) x 0    | 0      |
| 除算　　　　 | 正の数同士の除算ができる     | 6 / 2       | 3      |
| 除算         | 正の数と負の数の除算ができる | 6 / (-3)    | -2     |
| 除算         | 負の数同士の除算ができる     | (-6) / (-3) | 2      |
| 除算         | 0から整数を除算できる        | 0 / 5       | 0      |
