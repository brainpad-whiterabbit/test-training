# 演習スプリント 1

## スプリント概要

このスプリントでは、乗算と除算の機能を追加することを目指します。

## 要件レビュー

チュートリアルスプリント同様の手順でレビューを実施し、要件定義書にレビュー内容を記載してください。記載内容はcommit & pushしてください。

## テスト設計

チュートリアルスプリント同様の手順でテストケースを作成してください。テストケース表の変更内容はcommit & pushしてください。

| テスト観点 | テストケース | 入力 | 期待値 |
| :--- | :--- | :--- | :--- |
| 乗算 | 負数同士の乗算ができる | -2 * (-3) | 6 |
| 乗算 | 負数に正数を乗算できる | -2 * 3 | -6 |
| 乗算 | 正数に負数を乗算できる | 2 * (-3) | -6 |
| 乗算 | 正数同士の乗算ができる | 2 * 3 | 6 |
| 乗算 | 0を含む乗算ができる | 0 * 5 | 0 |
| 除算 | 負数同士の除算ができる | -6 / (-2) | 3 |
| 除算 | 負数から正数を除算できる | -6 / 2 | -3 |
| 除算 | 正数から負数を除算できる | 6 / (-2) | -3 |
| 除算 | 正数同士の除算ができる | 6 / 2 | 3 |
| 除算 | 0を含む除算（分子が0）ができる | 0 / 2 | 0 |
| 除算 | 0での除算（分母が0）でエラーになるか | 5 / 0 | ZeroDivisionError |
| 小数演算 | 負数（小数）同士の乗算ができる | -1.2 * (-2.0) | 2.4 |
| 小数演算 | 負数（小数）に正数（小数）を乗算できる | -1.2 * 1.5 | -1.8 |
| 小数演算 | 正数（小数）に負数（小数）を乗算できる | 1.5 * (-1.2) | -1.8 |
| 小数演算 | 正数（小数）同士の乗算ができる | 1.2 * 2.5 | 3.0 |
| 小数演算 | 0を含む乗算（小数）ができる | 0.0 * 1.5 | 0.0 |
| 小数演算 | 負数（小数）同士の除算ができる | -2.4 / (-1.2) | 2.0 |
| 小数演算 | 負数（小数）から正数（小数）を除算できる | -2.4 / 1.2 | -2.0 |
| 小数演算 | 正数（小数）から負数（小数）を除算できる | 2.4 / (-1.2) | -2.0 |
| 小数演算 | 正数（小数）同士の除算ができる | 2.4 / 1.2 | 2.0 |
| 小数演算 | 0を含む除算（分子が0）ができる | 0.0 / 1.2 | 0.0 |
| 小数演算 | 0での除算（分母が0）でエラーになるか | 1.2 / 0.0 | ZeroDivisionError |
