# 演習スプリント 1

## スプリント概要

このスプリントでは、乗算と除算の機能を追加することを目指します。

チュートリアルスプリント同様の手順でテストケースを作成してください。テストケース表の変更内容はcommit & pushしてください。

| テスト観点 | テストケース | 入力 | 期待値 |
| ---------- | ------------ | ---- | ------ |
| 乗算       | 負数同士の乗算ができる   | -1 * (-3)           | 3     |
| 乗算       | 負数から正数を乗算できる | -1 * 1              | -1     |

| 乗算       | 最小値を含む乗算ができる | -999999 * 1         | -999999 |
| 乗算       | 最大値同士の乗算ができる | 999999 * 999999     | Overflow |
| 乗算       | 最小値同士の乗算ができる | -999999 * (-999999) | Overflow |
| 除算       | 負数同士の除算ができる   | -4 ÷ (-1)           | 4        |
| 除算       | 負数から正数を除算できる | -1 ÷ 1              | -1       |
| 除算       | 正数から負数を除算できる | 1 ÷ (-1)            | -1     |
| 除算       | 正数同士の除算ができる   | 12 ÷ 4               | 3     |
| 除算       | 0を含む除算ができる      | 0 ÷ 1               | 0      |
| 除算       | 最大値を含む除算ができる | 999999 ÷ 1          | 999999   |
| 除算       | 最小値を含む除算ができる | -999999 ÷ 1         | -999999 |
| 除算       | 最大値同士の除算ができる | 999999 ÷ 999999     | 1      |
| 除算       | 最小値同士の除算ができる | -999999 ÷ (-999999) | 1        |
