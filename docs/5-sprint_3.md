## テスト設計

チュートリアルスプリント同様の手順でテストケースを作成してください。テストケース表の変更内容はcommit & pushしてください。

| テスト観点     | テストケース                                         | 入力                        | 期待値            |
| -------------- | ---------------------------------------------------- | --------------------------- | ----------------- |
| パーセント計算 | `%` 押下時に入力1に対する入力2の割合を計算できること | 左: 200, 右: 50, 操作: `%`  | 100 (200 の 50%)  |
| パーセント計算 | 小数値を含む入力でも割合を計算できること             | 左: 1.5, 右: 50, 操作: `%`  | 0.75 (1.5 の 50%) |
| パーセント計算 | 0% の場合に 0 を返すこと                             | 左: 100, 右: 0, 操作: `%`   | 0                 |
| パーセント計算 | 100% の場合に元の値を返すこと                        | 左: 100, 右: 100, 操作: `%` | 100               |
