# 演習スプリント 3

## スプリント概要

このスプリントでは、パーセント計算実行の機能を追加することを目指します。

## 要件レビュー

チュートリアルスプリント同様の手順でレビューを実施し、要件定義書にレビュー内容を記載してください。記載内容はcommit & pushしてください。

## テスト設計

チュートリアルスプリント同様の手順でテストケースを作成してください。テストケース表の変更内容はcommit & pushしてください。

| テスト観点                  | テストケース                                                                       | 入力                                         | 期待値                                                        |
| --------------------------- | ---------------------------------------------------------------------------------- | -------------------------------------------- | ------------------------------------------------------------- |
| パーセント計算実行 (F-17)   | 初期状態で「%」ボタンを押下した際、クリア状態（0）が維持されること                 | 起動直後に「%」を押下                        | 画面表示が "0" のまま維持される                               |
| パーセント計算実行 (F-17)   | 入力1のみの状態で「%」ボタンを押下した際、状態がクリアされること                   | "1", "0", "0" と入力した後に「%」を押下      | 画面表示が "0" にクリアされる                                 |
| パーセント計算実行 (F-17)   | 演算子のみ選択（入力2がない）状態で「%」ボタンを押下した際、状態がクリアされること | "1", "0", "0", "+" と入力した後に「%」を押下 | 画面表示が "0" にクリアされ、選択中の演算子も消える           |
| パーセント計算ルール (F-18) | 加算におけるパーセント計算が正しく実行されること                                   | 200 + 10 % を順番に押下                      | 画面表示が "220" になる（200 + (200 \* 10 / 100)）            |
| パーセント計算ルール (F-18) | 減算におけるパーセント計算が正しく実行されること                                   | 200 - 10 % を順番に押下                      | 画面表示が "180" になる（200 - (200 \* 10 / 100)）            |
| パーセント計算ルール (F-18) | 乗算におけるパーセント計算が正しく実行されること                                   | 200 x 10 % を順番に押下                      | 画面表示が "20" になる（200 \* (10 / 100)）                   |
| パーセント計算ルール (F-18) | 除算におけるパーセント計算が正しく実行されること                                   | 200 ÷ 10 % を順番に押下                      | 画面表示が "2000" になる（200 / (10 / 100)）                  |
| パーセント計算ルール (F-18) | 小数を含むパーセント計算が正しく実行されること                                     | 100 + 5.5 % を順番に押下                     | 画面表示が "105.5" になる（100 + (100 \* 5.5 / 100)）         |
| 特殊ケース（応用）          | パーセント計算で0除算が発生した場合にエラーが表示されること                        | 100 ÷ 0 % を順番に押下                       | 入力メッセージ欄に "Error" と表示される（100 / (0 / 100)）    |
| 特殊ケース（応用）          | パーセント計算の結果が表示範囲を超えた場合にエラーが表示されること                 | 999999 + 100 % を順番に押下                  | 入力メッセージ欄に "Overflow" と表示される（999999 + 999999） |
