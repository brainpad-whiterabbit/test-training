## チュートリアルスプリント

### 関連するテスト観点

- 減算
- 乗算
- 除算
- 小数演算

### 減算

- 入力値を同値分割
    - 最小値, 負数, 0, 正数, 最大値
- 右と左の入力値の組み合わせを同値分割
    - (負 - 負)
    - (負 - 正)
    - (正 - 負)
    - (正 - 正)
    - 0を含む
    - 最大値を含む
    - 最小値を含む
    - (最大値 - 最大値)
    - (最小値 - 最小値)

| テスト観点 | テストケース             | 入力          | 期待値   |
| ---------- | ------------------------ | ------------- | -------- |
| 減算       | 負数同士の減算ができる   | -1 - (-1)     | 0        |
| 減算       | 負数から正数を減算できる | -1 - 1        | -2       |
| 減算       | 正数から負数を減算できる | 1 - (-1)      | 2        |
| 減算       | 正数同士の減算ができる   | 1 - 1         | 0        |
| 減算       | 0を含む減算ができる      | 0 - 1         | -1       |
| 減算       | 最大値を含む減算ができる | Max - 1       | Max - 1  |
| 減算       | 最小値を含む減算ができる | -Max - 1      | Overflow |
| 減算       | 最大値同士の減算ができる | Max - Max     | 0        |
| 減算       | 最小値同士の減算ができる | -Max - (-Max) | 0        |

### 乗算

- 入力値を同値分割
    - 最小値, 負数, 0, 正数, 最大値
- 右と左の入力値の組み合わせを同値分割
    - (負 × 負)
    - (負 × 正)
    - (正 × 負)
    - (正 × 正)
    - 0を含む
    - 最大値を含む
    - 最小値を含む
    - (最大値 × 最大値)
    - (最小値 × 最小値)

| テスト観点 | テストケース                                 | 入力      | 期待値   |
| ---------- | -------------------------------------------- | --------- | -------- |
| 乗算       | 負数同士の乗算ができる                       | -1 × -1   | 1        |
| 乗算       | 負数と正数の乗算ができる                     | -1 × 1    | -1       |
| 乗算       | 正数と負数の乗算ができる                     | 1 × -1    | -1       |
| 乗算       | 正数同士の乗算ができる                       | 1 × 1     | 1        |
| 乗算       | 0を含む乗算ができる                          | 0 × 1     | 0        |
| 乗算       | 最大値を含む乗算ができる                     | Max × 1   | Max      |
| 乗算       | 最小値を含む乗算ができる                     | Min × 1   | Min      |
| 乗算       | 最大値と負数の乗算ができる                   | Max × -1  | -Max     |
| 乗算       | 最小値と負数の乗算ができる                   | Min × -1  | Overflow |
| 乗算       | 最大値同士の乗算でオーバーフローを検知できる | Max × Max | Overflow |
| 乗算       | 最小値同士の乗算でオーバーフローを検知できる | Min × Min | Overflow |

### 除算

- 入力値を同値分割
    - 最小値, 負数, 0, 正数, 最大値

- 左右の入力値の組み合わせを同値分割
    - (負 ÷ 負)
    - (負 ÷ 正)
    - (正 ÷ 負)
    - (正 ÷ 正)
    - 0を含む
    - 最大値を含む
    - 最小値を含む
    - (最大値 ÷ 最大値)
    - (最小値 ÷ 最小値)

- その他
    - 0で割った場合はエラーになる
    - 0を割られる数にした場合は結果が0になる
    - 割り切れる場合の結果を確認する
    - 割り切れない場合の結果を確認する
    - Min ÷ -1 はオーバーフローになる

| テスト観点 | テストケース                                         | 入力      | 期待値   |
| ---------- | ---------------------------------------------------- | --------- | -------- |
| 除算       | 負数同士の除算ができる                               | -1 ÷ -1   | 1        |
| 除算       | 負数を正数で除算できる                               | -1 ÷ 1    | -1       |
| 除算       | 正数を負数で除算できる                               | 1 ÷ -1    | -1       |
| 除算       | 正数同士の除算ができる                               | 1 ÷ 1     | 1        |
| 除算       | 0を正数で除算できる                                  | 0 ÷ 1     | 0        |
| 除算       | 0で割った場合にエラーになる                          | 1 ÷ 0     | Error    |
| 除算       | 最大値を含む除算ができる                             | Max ÷ 1   | Max      |
| 除算       | 最小値を含む除算ができる                             | Min ÷ 1   | Min      |
| 除算       | 最大値同士の除算ができる                             | Max ÷ Max | 1        |
| 除算       | 最小値同士の除算ができる                             | Min ÷ Min | 1        |
| 除算       | 最小値を-1で除算した場合にオーバーフローを検知できる | Min ÷ -1  | Overflow |
| 除算       | 割り切れない除算の結果を確認できる                   | 1 ÷ 2     | 0.5      |

### 小数演算

- 入力値を同値分割
    - 整数, 正の小数, 負の小数, 0, 非常に小さい小数, 非常に大きい小数
- 小数同士の組み合わせを最低限作成
    - 正の小数, 正の小数
    - 正の小数, 負の小数
    - 負の小数, 正の小数
    - 負の小数, 負の小数
    - 正の小数, 0
    - 0, 正の小数
    - 負の小数, 0
    - 0, 負の小数
    - 整数, 正の小数
    - 正の小数, 整数
    - 非常に小さい小数, 非常に小さい小数
    - 非常に大きい小数, 非常に大きい小数

| テスト観点 | テストケース                       | 入力                             | 期待値               |
| ---------- | ---------------------------------- | -------------------------------- | -------------------- |
| 小数演算   | 正の小数同士の演算ができる         | 1.5 + 2.5                        | 4.0                  |
| 小数演算   | 正の小数と負の小数の演算ができる   | 1.5 + (-0.5)                     | 1.0                  |
| 小数演算   | 負の小数と正の小数の演算ができる   | -1.5 + 0.5                       | -1.0                 |
| 小数演算   | 負の小数同士の演算ができる         | -1.5 + (-2.5)                    | -4.0                 |
| 小数演算   | 正の小数と0の演算ができる          | 1.5 + 0                          | 1.5                  |
| 小数演算   | 0と正の小数の演算ができる          | 0 + 1.5                          | 1.5                  |
| 小数演算   | 負の小数と0の演算ができる          | -1.5 + 0                         | -1.5                 |
| 小数演算   | 0と負の小数の演算ができる          | 0 + (-1.5)                       | -1.5                 |
| 小数演算   | 整数と正の小数の演算ができる       | 1 + 1.5                          | 2.5                  |
| 小数演算   | 正の小数と整数の演算ができる       | 1.5 + 1                          | 2.5                  |
| 小数演算   | 非常に小さい小数同士の演算ができる | 0.00000001 + 0.00000001          | 0.00000002           |
| 小数演算   | 非常に大きい小数同士の演算ができる | 9999999999.99999999 + 0.00000001 | 10000000000.00000000 |
