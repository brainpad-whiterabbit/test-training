# テストケース例

## 演算

### 加算

- 入力値を同値分割
    - 最小値, 負数, 0, 正数, 最大値
- 右と左の入力値の組み合わせ
    - (負 + 負)
    - (負 + 正)
    - (正 + 負)
    - (正 + 正)
    - 0を含む
    - 最大値を含む
    - 最小値を含む

| テスト観点 | テストケース             | 入力           | 期待値  |
| ---------- | ------------------------ | -------------- | ------- |
| 加算       | 負数同士の加算ができる   | -1 + (-1)      | -2      |
| 加算       | 負数と正数を加算できる   | -1 + 3         | 2       |
| 加算       | 正数と負数を加算できる   | 1 + (-3)       | -2      |
| 加算       | 正数同士の加算ができる   | 5 + 1          | 6       |
| 加算       | 0を含む加算ができる      | 0 + 1          | 1       |
| 加算       | 最大値を含む加算ができる | 999999 + (-10) | 999989  |
| 加算       | 最小値を含む加算ができる | -999999 + 10   | -999989 |

### 減算

- 入力値を同値分割
    - 最小値, 負数, 0, 正数, 最大値
- 右と左の入力値の組み合わせ
    - (負 - 負)
    - (負 - 正)
    - (正 - 負)
    - (正 - 正)
    - 0を含む
    - 最大値を含む
    - 最小値を含む
    - (最大値 - 最大値)
    - (最小値 - 最小値)
    - (最大値 + 最大値)
        - Overflowのテストケースにいれる

| テスト観点 | テストケース             | 入力                | 期待値   |
| ---------- | ------------------------ | ------------------- | -------- |
| 減算       | 負数同士の減算ができる   | -1 - (-3)           | 2        |
| 減算       | 負数から正数を減算できる | -1 - 1              | -2       |
| 減算       | 正数から負数を減算できる | 1 - (-1)            | 2        |
| 減算       | 正数同士の減算ができる   | 5 - 1               | 4        |
| 減算       | 0を含む減算ができる      | 0 - 1               | -1       |
| 減算       | 最大値を含む減算ができる | 999999 - 1          | 999998   |
| 減算       | 最小値を含む減算ができる | -999999 - 1         | Overflow |
| 減算       | 最大値同士の減算ができる | 999999 - 999999     | 0        |
| 減算       | 最小値同士の減算ができる | -999999 - (-999999) | 0        |

### 乗算

- 入力値を同値分割
    - 最小値, 負数, 0, 正数, 最大値
- 右と左の入力値の組み合わせ
    - (負 × 負)
    - (負 × 正)
    - (正 × 負)
    - (正 × 正)
    - 0を含む
    - 最大値を含む
    - 最小値を含む
    - (最大値 × 最大値)
        - Overflowのテストケースにいれる
    - (最小値 × 最小値)
        - Overflowのテストケースにいれる

| テスト観点 | テストケース             | 入力        | 期待値  |
| ---------- | ------------------------ | ----------- | ------- |
| 乗算       | 負数同士の乗算ができる   | -2 × -5     | 10      |
| 乗算       | 負数と正数の乗算ができる | -2 × 5      | -10     |
| 乗算       | 正数と負数の乗算ができる | 2 × -5      | -10     |
| 乗算       | 正数同士の乗算ができる   | 5 × 5       | 25      |
| 乗算       | 0を含む乗算ができる      | 0 × 5       | 0       |
| 乗算       | 最大値を含む乗算ができる | 999999 × 1  | 999999  |
| 乗算       | 最小値を含む乗算ができる | -999999 × 1 | -999999 |

### 除算

- 入力値を同値分割
    - 最小値, 負数, 0, 正数, 最大値

- 左右の入力値の組み合わせ
    - (負 ÷ 負)
    - (負 ÷ 正)
    - (正 ÷ 負)
    - (正 ÷ 正)
    - 0を含む
    - 最大値を含む
    - 最小値を含む
    - (最大値 ÷ 最大値)
    - (最小値 ÷ 最小値)
    - (最大値 ÷ 最小値)
    - 0での除算
        - エラーのテストケースにいれる

| テスト観点 | テストケース               | 入力              | 期待値  |
| ---------- | -------------------------- | ----------------- | ------- |
| 除算       | 負数同士の除算ができる     | -10 ÷ -2          | 5       |
| 除算       | 負数を正数で除算できる     | -10 ÷ 2           | -5      |
| 除算       | 正数を負数で除算できる     | 10 ÷ -2           | -5      |
| 除算       | 正数同士の除算ができる     | 10 ÷ 2            | 5       |
| 除算       | 0を正数で除算できる        | 0 ÷ 10            | 0       |
| 除算       | 最大値を含む除算ができる   | 999999 ÷ 9        | 111111  |
| 除算       | 最小値を含む除算ができる   | -999999 ÷ 9       | -111111 |
| 除算       | 最大値同士の除算ができる   | 999999 ÷ 999999   | 1       |
| 除算       | 最小値同士の除算ができる   | -999999 ÷ -999999 | 1       |
| 除算       | 最大値を最小値で除算できる | 999999 ÷ -999999  | -1      |

### Overflow

- 入力値を同値分割
    - 最小値, 負数, 0, 正数, 最大値
- 計算結果
    - 最大値を超える
    - 最小値を下回る
    - 乗算で表示桁数を超える
    - 小数部が表示桁数を超える

| テスト観点 | テストケース                         | 入力                 | 期待値   |
| ---------- | ------------------------------------ | -------------------- | -------- |
| Overflow   | 加算結果が最大値を超える             | 999999 + 1           | Overflow |
| Overflow   | 減算結果が最小値を下回る             | -999999 - 1          | Overflow |
| Overflow   | 正数同士の乗算結果が表示桁数を超える | 999999 × 999999      | Overflow |
| Overflow   | 負数同士の乗算結果が表示桁数を超える | -999999 × -999999    | Overflow |
| Overflow   | 正数と負数の乗算結果が最小値を下回る | 999999 × -999999     | Overflow |
| Overflow   | 小数部が4桁を超える結果は丸められる  | 1 ÷ 3                | 0.3333   |
| Overflow   | 丸め後も整数部が最大値以内なら表示   | 999999.9999 - 0.9999 | 999999   |

## 制限

- 入力値を同値分割
    - 整数部6桁以内
    - 整数部7桁以上
    - 小数部4桁以内
    - 小数部5桁以上
- 入力状態
    - 正数
    - 負数
    - 整数
    - 小数

| テスト観点   | テストケース                             | 入力操作            | 期待値  |
| ------------ | ---------------------------------------- | ------------------- | ------- |
| 表示桁数制限 | 整数部6桁まで入力できる                  | 999999              | 999999  |
| 表示桁数制限 | 整数部7桁目の入力を受け付けない          | 999999 に 9 を追加  | 999999  |
| 表示桁数制限 | 負数の整数部6桁まで入力できる            | -999999             | -999999 |
| 表示桁数制限 | 負数の整数部7桁目の入力を受け付けない    | -999999 に 9 を追加 | -999999 |
| 表示桁数制限 | 小数部4桁まで入力できる                  | 1.9999              | 1.9999  |
| 表示桁数制限 | 小数部5桁目の入力を受け付けない          | 1.9999 に 9 を追加  | 1.9999  |
| 表示桁数制限 | 整数部の上限超過時に表示中の値を維持する | 123456 に 7 を追加  | 123456  |
| 表示桁数制限 | 小数部の上限超過時に表示中の値を維持する | 0.1234 に 5 を追加  | 0.1234  |

### 計算結果桁数制限

- 計算結果を同値分割
    - 整数部6桁以内
    - 整数部7桁以上
    - 小数部4桁以内
    - 小数部5桁以上
- 結果表示
    - 表示可能な結果
    - Overflowになる結果
    - 丸められる結果

| テスト観点       | テストケース                             | 入力                 | 期待値   |

| 計算結果桁数制限 | 乗算結果が表示桁数を超える               | 999999 × 999999      | Overflow |
| 計算結果桁数制限 | 表示可能な最大値はOverflowにならない     | 999998 + 1           | 999999   |
| 計算結果桁数制限 | 表示可能な最小値はOverflowにならない     | -999998 - 1          | -999999  |
| 計算結果桁数制限 | 小数部が4桁以内の結果を表示できる        | 1.234 + 0.0005       | 1.2345   |
| 計算結果桁数制限 | 小数部が5桁以上の結果は4桁に丸められる   | 1 ÷ 3                | 0.3333   |
| 計算結果桁数制限 | 丸め後も表示桁数内ならOverflowにならない | 999999.9999 - 0.9999 | 999999   |

## パーセント計算

- 入力値を同値分割
    - 負数, 0, 正数, 最大値, 最小値
    - 整数, 小数
- パーセント値を同値分割
    - 負数, 0, 正数, 100, 100超, 小数
- 入力状態
    - 入力2
    - 入力2以外
- 入力2状態での演算子との組み合わせ
    - 加算, 減算, 乗算, 除算
- 計算結果
    - 表示桁数以内
    - 小数部が表示桁数を超える
    - 最大値を超える
    - 最小値を下回る
    - 計算不能

| -------------- | ------------------------------------------------------------ | ------------------- | -------- |
| パーセント計算 | 入力1に対する入力2の割合を加算して結果を表示する             | 100 + 10 %          | 110      |
| パーセント計算 | 入力1に対する0%を加算して結果を表示する                      | 100 + 0 %           | 100      |
| パーセント計算 | 入力1に対する100%を加算して結果を表示する                    | 100 + 100 %         | 200      |
| パーセント計算 | 入力1に対する100%超の割合を加算して結果を表示する            | 100 + 150 %         | 250      |
| パーセント計算 | 負数の入力1に対する割合を加算して結果を表示する              | -100 + 10 %         | -110     |
| パーセント計算 | 入力1に対する入力2の割合を減算して結果を表示する             | 100 - 10 %          | 90       |
| パーセント計算 | 入力1に対する100%を減算して結果を表示する                    | 100 - 100 %         | 0        |
| パーセント計算 | 入力1に対する100%超の割合を減算して結果を表示する            | 100 - 150 %         | -50      |
| パーセント計算 | 負数の入力1に対する割合を減算して結果を表示する              | -100 - 10 %         | -90      |
| パーセント計算 | 入力2の百分率を乗算して結果を表示する                        | 100 × 10 %          | 10       |
| パーセント計算 | 入力2の小数の百分率を乗算して結果を表示する                  | 200 × 12.5 %        | 25       |
| パーセント計算 | 負数に入力2の百分率を乗算して結果を表示する                  | -100 × 10 %         | -10      |
| パーセント計算 | 入力2の百分率で除算して結果を表示する                        | 100 ÷ 10 %          | 1000     |
| パーセント計算 | 負数を入力2の百分率で除算して結果を表示する                  | -100 ÷ 10 %         | -1000    |
| パーセント計算 | 入力1に対する0%で除算するとErrorになる                       | 100 ÷ 0 %           | Error    |

| パーセント計算 | パーセント加算結果が最大値を超えるとOverflowになる           | 999999 + 1 %        | Overflow |
| パーセント計算 | 負数基準のパーセント加算結果が最小値を下回るとOverflowになる | -999999 + 1 %       | Overflow |
