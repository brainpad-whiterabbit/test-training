# テストケース例

## 加算

- 入力値を同値分割
    - 最小値, 負数, 0, 正数, 最大値
- 右と左の入力値の組み合わせ
    - (負 + 負)
    - (負 + 正)
    - (正 + 負)
    - (正 + 正)
    - 0を含む
    - 最大値を含む
    - 最小値を含む

| テスト観点 | テストケース             | 入力           | 期待値  |
| ---------- | ------------------------ | -------------- | ------- |
| 加算       | 負数同士の加算ができる   | -1 + (-1)      | -2      |
| 加算       | 負数と正数を加算できる   | -1 + 3         | 2       |
| 加算       | 正数と負数を加算できる   | 1 + (-3)       | -2      |
| 加算       | 正数同士の加算ができる   | 5 + 1          | 6       |
| 加算       | 0を含む加算ができる      | 0 + 1          | 1       |
| 加算       | 最大値を含む加算ができる | 999999 + (-10) | -999989 |
| 加算       | 最小値を含む加算ができる | -999999 + 10   | -999989 |

## 減算

- 入力値を同値分割
    - 最小値, 負数, 0, 正数, 最大値
- 右と左の入力値の組み合わせ
    - (負 - 負)
    - (負 - 正)
    - (正 - 負)
    - (正 - 正)
    - 0を含む
    - 最大値を含む
    - 最小値を含む
    - (最大値 - 最大値)
    - (最小値 - 最小値)
    - (最大値 + 最大値)
        - Overflowのテストケースにいれる

| テスト観点 | テストケース             | 入力                | 期待値   |
| ---------- | ------------------------ | ------------------- | -------- |
| 減算       | 負数同士の減算ができる   | -1 - (-3)           | 2        |
| 減算       | 負数から正数を減算できる | -1 - 1              | -2       |
| 減算       | 正数から負数を減算できる | 1 - (-1)            | 2        |
| 減算       | 正数同士の減算ができる   | 5 - 1               | 4        |
| 減算       | 0を含む減算ができる      | 0 - 1               | -1       |
| 減算       | 最大値を含む減算ができる | 999999 - 1          | 999998   |
| 減算       | 最小値を含む減算ができる | -999999 - 1         | Overflow |
| 減算       | 最大値同士の減算ができる | 999999 - 999999     | 0        |
| 減算       | 最小値同士の減算ができる | -999999 - (-999999) | 0        |

## 乗算

- 入力値を同値分割
    - 最小値, 負数, 0, 正数, 最大値
- 右と左の入力値の組み合わせ
    - (負 × 負)
    - (負 × 正)
    - (正 × 負)
    - (正 × 正)
    - 0を含む
    - 最大値を含む
    - 最小値を含む
    - (最大値 × 最大値)
        - Overflowのテストケースにいれる
    - (最小値 × 最小値)
        - Overflowのテストケースにいれる

| テスト観点 | テストケース             | 入力        | 期待値  |
| ---------- | ------------------------ | ----------- | ------- |
| 乗算       | 負数同士の乗算ができる   | -2 × -5     | 10      |
| 乗算       | 負数と正数の乗算ができる | -2 × 5      | -10     |
| 乗算       | 正数と負数の乗算ができる | 2 × -5      | -10     |
| 乗算       | 正数同士の乗算ができる   | 5 × 5       | 25      |
| 乗算       | 0を含む乗算ができる      | 0 × 5       | 0       |
| 乗算       | 最大値を含む乗算ができる | 999999 × 1  | 999999  |
| 乗算       | 最小値を含む乗算ができる | -999999 × 1 | -999999 |

## 除算

- 入力値を同値分割
    - 最小値, 負数, 0, 正数, 最大値

- 左右の入力値の組み合わせ
    - (負 ÷ 負)
    - (負 ÷ 正)
    - (正 ÷ 負)
    - (正 ÷ 正)
    - 0を含む
    - 最大値を含む
    - 最小値を含む
    - (最大値 ÷ 最大値)
    - (最小値 ÷ 最小値)
    - (最大値 ÷ 最小値)
    - 0での除算
        - エラーのテストケースにいれる

| テスト観点 | テストケース               | 入力              | 期待値  |
| ---------- | -------------------------- | ----------------- | ------- |
| 除算       | 負数同士の除算ができる     | -10 ÷ -2          | 5       |
| 除算       | 負数を正数で除算できる     | -10 ÷ 2           | -5      |
| 除算       | 正数を負数で除算できる     | 10 ÷ -2           | -5      |
| 除算       | 正数同士の除算ができる     | 10 ÷ 2            | 5       |
| 除算       | 0を正数で除算できる        | 0 ÷ 10            | 0       |
| 除算       | 最大値を含む除算ができる   | 999999 ÷ 9        | 111111  |
| 除算       | 最小値を含む除算ができる   | -999999 ÷ 9       | -111111 |
| 除算       | 最大値同士の除算ができる   | 999999 ÷ 999999   | 1       |
| 除算       | 最小値同士の除算ができる   | -999999 ÷ -999999 | 1       |
| 除算       | 最大値を最小値で除算できる | 999999 ÷ -999999  | -1      |

## 小数演算

- 入力値を同値分割
    - 整数, 正の小数, 負の小数, 0
- 演算子
    - 加算, 減算, 乗算, 除算
- 小数同士の組み合わせを最低限作成
    - 正の小数, 正の小数, 加算
    - 正の小数, 正の小数, 減算
    - 正の小数, 正の小数, 乗算
    - 正の小数, 正の小数, 除算
    - 正の小数, 負の小数
    - 負の小数, 負の小数
    - 正の小数, 0
    - 整数, 正の小数
    - 非常に大きい小数, 非常に小さい小数

| テスト観点 | テストケース                                     | 入力                 | 期待値      |
| ---------- | ------------------------------------------------ | -------------------- | ----------- |
| 小数演算   | 正の小数同士の加算ができる                       | 1.5 + 2.5            | 4.0         |
| 小数演算   | 正の小数同士の減算ができる                       | 1.5 - 2.5            | -1.0        |
| 小数演算   | 正の小数同士の乗算ができる                       | 1.5 × 2.5            | 3.75        |
| 小数演算   | 正の小数同士の除算ができる                       | 1.5 ÷ 2.5            | 0.6         |
| 小数演算   | 正の小数と負の小数の演算ができる                 | 1.5 + (-0.5)         | 1.0         |
| 小数演算   | 負の小数と正の小数の演算ができる                 | -1.5 + 0.5           | -1.0        |
| 小数演算   | 負の小数同士の演算ができる                       | -1.5 + (-2.5)        | -4.0        |
| 小数演算   | 正の小数と0の演算ができる                        | 1.5 + 0              | 1.5         |
| 小数演算   | 整数と正の小数の演算ができる                     | 1 + 1.5              | 2.5         |
| 小数演算   | 正の小数と整数の演算ができる                     | 1.5 + 1              | 2.5         |
| 小数演算   | 非常に大きい小数と非常に小さい小数の演算ができる | 999999.9999 - 0.0001 | 999999.9998 |
