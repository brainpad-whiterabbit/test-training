# 観点資料

## 要件レビュー観点

以下をレビュー観点とします。

- 曖昧な表現や誤りが含まれていないか
    - 曖昧な例: 「適切に」, 「簡単に」など
- 要件内容について、主体、入力、処理、出力、条件が明確に理解できるように記載されているか
    - 理解できれば必ずしも含めなくてもよい
- 第三者でも理解できて簡潔な文章になっているか
- 記載粒度が統一されているか
- 用語の使い方が統一されているか
- 未記載・記載漏れがないか
- 要件同士で矛盾がないか
- テスト可能か
